# LED-KEIKOKAN

BLEのパケットを自前で作成するのは断念

- パケットキャプチャの結果、カウンタ部分以外はほぼ暗号化されているように見えた
- まったく同じパケットをラズパイから送信した場合、対抗機の電源投入直後の1回だけ操作できた。
- しかし2回目以降は操作不可であったことから、リプレイ対策もされていると考えられる。
- 基本的にBeacon的な通信が行われており、ペアリング用のsmp通信も見えなかったことから解析の取っ掛かりなし
- 基板上のUARTっぽいPADもオシロ・ロジアナでチェックしたが意味のある文字列出力は見受けられず。
- Codexに片っ端からUARTのボーレートやパリティの有無などを試してもらったが結果は変わらず。
- 開発時の治具か最終検査用か何かの名残の出力だろうと判断

次に試すなら、物理リモコン自体を外部から操作する手段。  
3Dプリンタで筐体用意すれば基板を隠したり操作機と同梱させたりできそうだが、その時点でスマートでないので悩みどころ。
